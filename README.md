# Project Title
# Data Cleaning and Preprocessing using Python (Pandas)
# Objective
The objective of this project is to clean and prepare a raw dataset by handling missing values, removing duplicates, standardizing data formats, and ensuring consistency for analysis.
# Dataset
 Dataset Name: Netflix Movies and TV Shows
 Source: Kaggle
 Format: CSV file
# Tools Used
Python
Pandas Library
Jupyter Notebook
# Steps Performed:
1. Handling Missing Values: Identified null values using .isnull() and filled them using .fillna().
2. Removing Duplicates: Removed duplicate rows using .drop_duplicates().
3. Standardizing Text: Cleaned and formatted text fields like country and type.
4. Date Conversion: Converted date_added column into datetime format.
5. Column Renaming: Renamed columns to lowercase and removed spaces.
6. Fixing Data Types: Ensured proper data types for columns
# Summary:
The dataset was cleaned by handling missing values, removing duplicates, standardizing text, and
fixing formats. The final dataset is ready for analysis
